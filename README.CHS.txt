新未来世界项目 (基于Shoebill)
=================
本项目是基于原未来世界理念而全新设计的新地图。
使用 Java 和 Shoebill API 完全重新设计和制作的新未来世界。

项目和原版未来世界没有任何直接的关系，但是依旧感谢原版未来世界的所有贡献者的无私贡献。
[点击这里](http://www.gtabbs.com/read-gta-tid-2593634.html) 可以访问到原版未来世界的发布帖。

总体设计
-----------------
* 重视人性化设计。
* 以设计强大、华丽的系统为目标。
* 重视数据跟踪和回报能力。
* 尽量提供所有用户生成的数据的再编辑能力。
* 尽量设计具有动态更新和适应的能力的功能。
* 强力而完善的多语言支持。
* 使用智能而又便利的对话框作为主要用户交互界面，命令操作只作为快捷功能和辅助使用。
* 主要的对话框均设计为可拓展式，根据启用的插件添加相关选择项。
* 使用快捷键简化最常用的操作。
* （汉语支持）简繁自动编码转换功能（基于 Shoebill）。

开发
-----------------
* 保证代码的质量和高可读性。
* 项目将会分拆成多个组件，可以自行决定部分非必须组件是否启用（支持动态开启/关闭）。
* 所有组件都提供交互接口，可与其他组件互相协作。
* 使用 YAML 作为配置文件的存储格式。
* 使用 MongoDB 数据库 (和 Morphia 对象映射库) 来存储数据。

组件
-----------------
目前以下组件已被计划开发：
* `wl-gamemode` 主模块，主要负责协调各组件的工作
* `wl-common` 项目专属通用工具库
* `wl-auth` 账号认证和权限系统
* `wl-chat-channel` 聊天频道系统
* `wl-vehicle-manager` 车辆管理系统
* `wl-teleport` 传送点和世界切换系统
* `wl-race` 高级赛车系统
* `wl-language` 多语言支持
* `wl-radio` 在线电台系统
* `wl-admin` 管理员管理系统
* `wl-simple-dm` 简易DM系统
* `wl-anticheat` 简易反作弊系统
* `wl-mail` 邮件系统

依赖的组件和开源项目
-----------------
* `shoebill-common` Shoebill 通用工具库
* `common-lang3` Apache Java 通用工具库
* `mongo-java-driver` Mongo DB Java 数据库驱动
* `morphia` 对象映射库

推荐使用的组件
-----------------
* `mk-plugin-manager` MK 的 Shoebill 插件管理器
